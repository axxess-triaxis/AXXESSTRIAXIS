# Privacy Data Map

AXXESS stores and processes:

- Account profile data.
- Organization, department, workspace, role, and invitation records.
- Project, task, meeting, document metadata, and Knowledge Hub records.
- Private document storage objects.
- Audit logs.
- RAG source citations and AI output audit metadata.
- Sanitized analytics events.

AXXESS does not intentionally send raw document body text, prompts, secrets, passwords, or service-role credentials to analytics.

Data deletion requires review for:

- Audit retention.
- Legal hold.
- Shared tenant records.
- Storage objects.
- Vector/search indexes.
- Analytics identifiers.
