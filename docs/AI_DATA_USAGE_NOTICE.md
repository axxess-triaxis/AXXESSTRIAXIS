# AI Data Usage Notice

AXXESS uses governed RAG and local NLP to help users retrieve and summarize authorized institutional knowledge.

AI outputs may include:

- Source citations.
- Confidence score.
- Human review status.
- Prompt version evidence.
- Audit log references.

High-impact or low-confidence outputs require human review before operational use.

AXXESS should not retrieve documents outside the user's tenant, department, workspace, role, visibility, or explicit permissions.
