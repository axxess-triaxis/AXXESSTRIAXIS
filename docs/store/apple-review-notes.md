# Apple Review Notes

## App

AXXESS by Triaxis is an enterprise governance and institutional intelligence platform for organizations that need human review, audit evidence, document-grounded AI, approvals, project execution and operational command-center workflows.

## Reviewer Access

- Reviewer email: `reviewer.mobile@axxess.demo`
- Reviewer role: Organization Admin
- Demo tenant: North East Health Mission
- Authentication path: `/auth/login`

The reviewer account is intended for App Review and TestFlight review only. It opens a populated institutional preview tenant and does not grant access to production customer data.

## Review Path

1. Sign in with the reviewer account.
2. Open Executive Dashboard to review live institutional metrics.
3. Open AI Workspace and AI Review Inbox to inspect cited human-in-the-loop AI outputs.
4. Open Knowledge Hub to review governed institutional documents.
5. Open Approvals and Workflow Records to inspect auditable action creation.
6. Open Admin > Mobile Store Launch Console to inspect release readiness, screenshots, reviewer account state, crash health and staged rollout controls.

## Sensitive Capabilities

AXXESS does not perform consequential actions without a human review step. The mobile shell uses the production web runtime and respects tenant role-based access controls. Demo data is isolated from production tenants.
