# Beta Disclaimer

AXXESS is currently beta software.

Beta limitations may include:

- Provider-gated MFA, OAuth, and passkeys.
- Staging-only migration validation until Supabase credentials are applied.
- Mobile signing and store submission pending Apple/Google credentials.
- Compliance controls designed as an audit-ready foundation, not certification.

Do not represent beta readiness as SOC 2, ISO 27001, HIPAA, GDPR, DPDP, or government certification.
