# App Store Connect Setup

1. Confirm the Apple Developer Program membership and team.
2. Register the bundle identifier com.triaxis.axxess.
3. Create or validate the App Store distribution certificate.
4. Create or validate the provisioning profile for the App ID.
5. Configure an App Store Connect API key, then store the key ID, issuer ID, and private key in the secure credential store.
6. Only upload to TestFlight or the App Store after the required agreements are accepted.
